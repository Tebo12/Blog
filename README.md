# Blog API

Проект для курса по веб-разработке. Реализация бэкенда и простого фронтенда для блога.

## Технологии
* Python 3.10+
* FastAPI
* SQLAlchemy (Asyncio) + Alembic
* SQLite (для разработки)
* Docker (для деплоя с PostgreSQL)
* Jinja2 шаблоны
* Pytest

## Функционал
1. **Пользователи**: Регистрация, вход, профиль, редактирование данных.
2. **Посты**: Создание, просмотр, редактирование, удаление.
3. **Поиск**: Поиск постов по заголовку и тексту.
4. **Избранное**: Можно добавлять посты в избранное.
5. **Безопасность**: Хеширование паролей, авторизация через Cookie/JWT.

## Запуск

### Локально
1. Создать виртуальное окружение:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Применить миграции БД:
   ```bash
   alembic upgrade head
   ```
4. Запустить сервер:
   ```bash
   uvicorn app.main:app --reload
   ```
   Сайт будет доступен по адресу: http://127.0.0.1:8000

### Через Docker
```bash
docker-compose up --build
```

## Тесты
Для запуска тестов используется pytest:
```bash
pytest
```
